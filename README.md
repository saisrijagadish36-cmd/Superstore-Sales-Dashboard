Superstore Sales Dashboard

## Project Overview 
This project analyzes the Superstore sales dataset from Kaggle.
The goal is to find key business insights using Python and 
visualize them using Power BI Dashboard.

## Tools Used
- **Python** (Google Colab) — Data Cleaning & Analysis
- **Pandas Library** — Data manipulation
- **Power BI Desktop** — Dashboard & Visualization

## Dashboard Preview
![Dashboard](superstore_dashboard.png)

## Key Business Insights
- Total Sales: **$2.30M**
- Total Profit: **$286K**
- Best Category: **Technology ($836K)**
- Best Region: **West ($725K)**
- Biggest Loss: **Tables (-$17.7K)**
- Sales Growth: **51% from 2014 to 2017**
- Most Used Shipping: **Standard Class (60%)**
- Top Segment: **Consumer ($1.16M)**

## Project Steps
1. Downloaded Superstore dataset from Kaggle
2. Loaded and cleaned data using Python in Google Colab
3. Added new columns — Year, Month, Profit Margin
4. Analyzed KPIs, Categories, Regions, Sub-Categories
5. Exported clean CSV to Power BI
6. Built interactive dashboard in Power BI
7. Published to GitHub Portfolio
